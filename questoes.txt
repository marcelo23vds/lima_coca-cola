1 O que é um repositório no Git e qual a sua importância?
- um local para armazenar os projetos criados, pode ser util para versionamento de código para equipes ou comunidades e também funciona como portfolio

2 Qual a função do Git Bash?
- terminal do git para poder fazer commit, push, pull, etc

3 O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
- serve para selecionar os arquivos que vao para o commit, o . seleciona tudo e também podemos selecionar um <arquivo> em especifico

4 Por que utilizamos o ponto (.) no comando git add .?
- selecionar tudo (todos os arquivos) para o commit

5 Explique o comando git commit e por que é importante criar uma mensagem descritiva.
- é como criar um "checkpoint" em uma linha do tempo, onde caso precise voce pode voltar para uma versao anterior do projeto, e a mensagem bem explicada é importante para se localizar na branch

6 Qual a função do comando git push?
- enviar os arquivos que ja foram "commitados" para o repositório

7 Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
- Ter varias versoes ao longo da criacao do projeto, e poder realizar manutencoes sem quebrar o projeto

8 Explique a função das tags HTML e como elas estruturam uma página.
- Elas funcionam para referenciar a funcao de cada parte da pagina, a estrutura basica é feita com head e body

9 O que são atributos em uma tag HTML e qual a sua função?
- os atributos podem ser definidos como informacoes que uma tag necessita, por exemplo criar uma class para uma tag, voce vai poder formatar em css como deseja que fique aquela tag

10 O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
- ele facilita escrever codigo html porque preenche automatico muitas tags, temos um grande ganho de tempo com ele

11 O que é o box-sizing no CSS e por que usamos ele no reset inicial?
- Define como sao calculadas a width e height dos elementos, ela é usada para alterar a propriedade padrão da box model. 

12 Por que é importante utilizar tags semânticas no HTML?
- Para quando precisar fazer alteracoes ficar legivel e facil de entender

13 O que são media queries no CSS e qual a sua função?
- é utilizado para adaptar o conteudo de uma pagina para diferentes tipos de tela

14 Qual a função do GitHub Pages?
- Criar um local na web para poder acessar o projeto que criamos

15 O que é o Flexbox no CSS e para que ele é utilizado?
- para redimensionar em diferentes posicoes os elementos

16 Qual a diferença entre justify-content e align-items em um container Flexbox?
- Justify content alinha os elementos na horizontal e align items na vertical

17 Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
- Definir a largura de um elemento em relacao aos outros do mesmo container

18 Explique a diferença entre flex-direction: column; e flex-direction: row;.
- Vai definir se os itens do container devem seguir em colunas ou em linhas
